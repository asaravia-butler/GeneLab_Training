# Process RNA Sequence (RNAseq) Data Using Jupyter Notebooks (JNs)

The JN in this directory contains instructions and commands for processing raw RNA sequence files from [GLDS-104]() spaceflight and ground control samples using the GL pipeling blah from raw sequence files  The JNs and respective input files listed below must be in a location that's accessible within the JupyterHub web interface, https://domain/jupyter/. Completed RNAseq JNs in html format can be found in the [Example_RNAseq_JNs](Example_RNAseq_JNs) subdirectory.

## Jupyter Notebooks
- [RNAseq_fastq_to_counts_JN](RNAseq_fastq_to_counts_JN_06-2021.ipynb)
- [RNAseq_DGE_JN](RNAseq_DGE_JN_06-2021.ipynb)
